# AI Image Classification using AWS SageMaker and S3

## Project Overview
This project performs AI-based image classification using AWS SageMaker and Amazon S3.

## AWS Services Used
- Amazon SageMaker
- Amazon S3
- IAM

## Features
- Upload images
- AI predicts image category
- Uses pretrained ResNet model

## Technologies
- Python
- PyTorch
- AWS SageMaker
- Amazon S3

## Sample Predictions
- cat.jpg → tabby cat
- puppy.jpg → Labrador retriever

## Learning Outcome
Learned cloud-based AI workflow using AWS services.
